# php-practice4

이 프로젝트는 이전 작업을 기반으로, 게시판의 CRUD 기능과 보안 기능을 결합하여 개발한 웹 애플리케이션입니다.  
UI 디자인은 국내 최대 커뮤니티 중 하나인 **디시인사이드**의 스타일을 참고했으며,  
일부 기능도 직접 구현하여 유사한 사용자 경험을 제공하도록 구성했습니다. (CSS만 참고, 기능은 자체 구현)

## 구현된 주요 기능
- 로그인 / 회원가입 / 비밀번호 찾기
- 게시물 조회, 작성, 수정, 삭제
- 댓글 작성, 삭제 / 대댓글 작성
- 마이페이지 게시물, 댓글 조회
- 비로그인 사용자 이용 대응 / 캡차 시스템
- 게시물 추천, 비추천
- 게시물 검색
- 보안 취약점(SQL Injection, CSRF 등) 대응책 마련
